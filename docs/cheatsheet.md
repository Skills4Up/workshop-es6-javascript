
### Logica Assignment (&&=, ||=, ?==)

```js
// a &&= b equivale a: a = a && b
let x = 1;
x &&= 2; // x = 2

// a ||= b equivale a: a = a || b
let y = 0;
y ||= 5; // y = 5

// a ??= b equivale a: a = a ?? b
let z = undefined;
z ??= 10; // z = 10

let w = 0;
w ??= 42; // w = 0 (porque 0 no es null/undefined)
```
